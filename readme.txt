SweetMo WordPress Theme
---------------------
Created by Derrick Rice, rice.io


Install
-------
1. Install the "omega" parent theme. See http://wordpress.org/themes/omega for details.
2. Install GitHub Updater (https://github.com/afragen/github-updater)
3. Using GitHub Updater, install from https://github.com/DerrickRice/sweetmo_theme
